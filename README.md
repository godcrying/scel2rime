# 搜狗词库转文本格式
```
Convert sogou scel file to rime dict file.

USAGE:
    scel2rime [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -i, --input <input>      A sogou scel filename.
    -o, --output <output>    An output filename.
```
